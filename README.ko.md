# 실전 타자연습 (Arcade Typing)

[Read in English](./README.md)

**실전 타자연습**은 20세기 오락실 감성의 레트로 아케이드 타자 게임입니다. 단순한 타자 연습을 넘어, 물리 엔진이 적용된 단어들이 쌓이고 부딪히는 긴장감 넘치는 플레이를 제공합니다.

# Our Philosophy
IDDQD Internet은 별도의 DB나 회원가입 없이, 순수 HTML/JS만으로 브라우저에서 즉시 실행되는 도구를 개발합니다. AI 기능을 제공할 때도 데이터 상태를 유지하지 않으며(stateless), 어떠한 기록도 남기지 않는 원칙을 고수합니다.

### [실전 타자연습 플레이하기](https://game.iddqd.kr/typing)

## Features
- **물리 엔진 적용**: 단어들이 단순히 떨어지는 것이 아니라, 서로 부딪히고 쌓이는 물리 효과(Custom Physics)가 적용되어 게임의 깊이를 더합니다.
- **레트로 아케이드 비주얼**: CRT 모니터 효과, 스캔라인, 네온 타이포그래피로 90년대 오락실의 분위기를 재현했습니다.
- **다양한 매운맛 스테이지**:
    - **트렌드 신조어**: 요즘 유행하는 인싸 용어 완전 정복.
    - **김성모 명대사**: "더 이상의 자세한 설명은 생략한다" 등 전설적인 명대사.
    - **실전 아파트 청약**: 10자부터 100자까지 이어지는, 끔찍하게 긴 아파트 이름들의 향연.
- **로컬 인증서 발급**: 게임이 끝나면 자신의 기록이 담긴 '키보드 파괴 인증서' 이미지를 즉석에서 생성하고 저장할 수 있습니다.

## Usage
**목표**: 화면이 꽉 차기 전에 단어를 입력하여 파괴하세요!

1. **스테이지 선택**: 원하는 테마의 스테이지를 선택합니다.
2. **타이핑 & 파괴**:
    - 떨어지는 단어를 정확히 입력합니다.
    - **Enter** 키를 눌러 발사합니다.
    - **Tip**: 띄어쓰기는 무시해도 입력 인정됩니다!
3. **생존**: 레벨이 오를수록 속도가 빨라집니다. 단어가 화면 상단까지 쌓이면 게임 오버입니다.
4. **결과**: 점수에 따라 '독수리 타법'부터 '전설의 타자왕'까지 다양한 칭호가 부여됩니다.

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (jQuery)
- **UI**: Bootstrap 5
- **Audio**: Web Audio API (신디사이저 효과음), HTML5 Audio (BGM)
- **Physics**: Custom AABB Physics Logic
- **Tools**: html2canvas (인증서 이미지 생성)
